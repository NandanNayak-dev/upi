# UPI Clone App

## Overview

UPI Clone App is a simple digital payment application developed to simulate the basic functionalities of a UPI-based payment system. The application provides users with a clean and user-friendly interface to perform essential payment operations such as sending money, receiving money, and viewing transaction details.

This project was created to understand payment application workflows and strengthen practical knowledge of frontend and backend development concepts.

## Features

* User registration and login
* Secure authentication
* Send money to users
* Receive money
* View transaction history
* Simple and responsive user interface
* Basic wallet/balance management
